# Statistics: Inference and Modeling (R)

This repository contains practical R exercises completed as part of my training in **Statistical Inference and Modeling**.

The objective of this repository is to demonstrate the practical application of statistical concepts using **R** and the **tidyverse** ecosystem through real-world datasets and coding exercises.

## Topics Covered

- Sampling distributions
- Population vs. sample statistics
- Standard error
- Confidence intervals
- Hypothesis testing
- p-values
- Central Limit Theorem
- Monte Carlo simulations
- Statistical modeling
- Polling bias analysis
- Data visualization with ggplot2

## Technologies

- R
- tidyverse
- dplyr
- ggplot2
- dslabs

## Learning Objectives

The purpose of these exercises is to strengthen my understanding of statistical inference by implementing theoretical concepts in R, including:

- constructing confidence intervals;
- estimating sampling variability;
- performing hypothesis tests;
- interpreting p-values;
- using Monte Carlo simulations;
- analyzing real polling data.

## Dataset

Most exercises use datasets provided by the **dslabs** package developed for the HarvardX Data Science Professional Certificate.
